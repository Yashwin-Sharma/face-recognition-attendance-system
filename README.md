# Face Recognition Attendance System

This project is a Python-based face recognition system that automatically marks attendance using computer vision.

## Project Overview
The system detects and recognizes faces in real-time and records attendance with timestamps.

## Features
- Face detection and recognition
- Automatic attendance marking
- Timestamp storage using datetime module
- Error handling
- Accuracy optimization

## Technologies Used
- Python
- OpenCV
- face_recognition
- NumPy
